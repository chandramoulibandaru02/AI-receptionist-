"# AI receptionist project" 
done with the translation layer, chat history (memory and the memory will be stored up to past 20 messages), and the core conversational flow The bot now understands user input, responds in the same language, and keeps track of the conversation so it feels continuous. It can handle both hotel and restaurant queries properly, like giving food details, suggesting services, and guiding users toward actions such as booking a table or asking about rooms and pricing. I’ve used the Groq API with the gpt-oss-120b model, which is a strong pre-trained model, and I’ve constrained it so it stays focused only on hotel and restaurant use cases. If someone asks something unrelated, it politely redirects instead of answering. I’ve tested the translation mainly with Telugu so far and it’s working well.
